# Choice Card Screenshots

## How to Update

Simply replace these files with your gameplay screenshots:

- `choice1.jpg` - Cautious choice gameplay
- `choice2.jpg` - Bold choice gameplay  
- `choice3.jpg` - Free will/custom action gameplay

## Image Specs

- **Format**: JPG or PNG
- **Recommended size**: 800x450px (16:9 aspect ratio)
- **Max file size**: Keep under 500KB for fast loading

## Tips

1. **Use dramatic moments** - Tense scenes work best
2. **First-person POV** - Shows the player's perspective
3. **Clear composition** - The dark overlay will cover most details, so use high-contrast images
4. **Text is overlaid** - The bottom third will have text, so avoid important visual elements there

## After Updating

1. Save your screenshot as `choice1.jpg`, `choice2.jpg`, or `choice3.jpg`
2. Push to GitHub: `git add static/images/choices; git commit -m "Update choice screenshots"; git push`
3. Render will auto-deploy in ~2 minutes

That's it! The design automatically handles the rest.
