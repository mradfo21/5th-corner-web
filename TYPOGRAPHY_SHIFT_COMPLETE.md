## üîß KEY ADJUSTMENTS

### 1. Removed Bold Everywhere
**Why:** OCR and institutional fonts should not be bold.  
**Effect:** Feels utilitarian, not designed.

### 2. Increased Letter Spacing
**Why:** Government/archival material has mechanical spacing.  
**Effect:** Feels stamped, typed, scanned.

### 3. Reduced Glow Intensity
**Before:** `0 0 20px, 0 0 40px`  
**After:** `0 0 15px, 0 0 30px`  
**Why:** Less "neon sign", more "emergency lighting".

### 4. Smaller System Text
**Before:** 0.85rem  
**After:** 0.75rem  
**Why:** Machine status should be quiet, not dramatic.

### 5. Tighter Subtitle
**Before:** 0.15em spacing  
**After:** 0.08em spacing  
**Why:** DIN Condensed style - tight, industrial.
